# PlaylistPal

**PlaylistPal** is an ultimate playlist manager designed to help users organize and enjoy their music collection. Users can create, update, delete, and manage playlists, tracks, and their profile with ease.

## Features

- **User Authentication**: Secure login and signup with username, email, and password.
- **Profile Management**: Edit user profile details, including username, email, password, and profile image.
- **Playlist Management**: Create, update, and delete playlists.
- **Track Management**: Add, update, and delete tracks within playlists.
- **Responsive UI**: The app is built with Bootstrap to ensure itâ€™s mobile-friendly.

## Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: SQLite (Sequelize ORM)
- **Frontend**: HTML, CSS, Bootstrap, Handlebars.js
- **Authentication**: Sessions with Express-session
